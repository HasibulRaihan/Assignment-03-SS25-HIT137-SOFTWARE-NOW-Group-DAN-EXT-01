# Assignment-03-SS25-HIT137-SOFTWARE-NOW-Group-DAN-EXT-01
A simple Python-based Image Editor built with Tkinter. Includes a full GUI with menu bar, image display area, control panel for effects, adjustable sliders, status bar, and support for opening/saving JPG, PNG, and BMP files. Features undo/redo, file dialogs, and message boxes for confirmations and errors.
